<!--more-->

在命令行里输入 

```shell
unset SESSION_MANAGER&& netease-cloud-music
```

 可以运行。

不过我无法在 desktop 文件里用这个命令，所以只能把命令写一个 shell 文件然后用 sh 运行去运行了。亲测 Debian sid 的 cinnamon 里可用。望周知。
