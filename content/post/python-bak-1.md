## os
``` python
os.listdir() # 列出当前目录下的所有文件，相当于返回 ls 的结果

os.chdir(path) # 切换当前目录

os.path.splittext(path) # 分离文件和扩展名

os.path.exits(path) # 路径是否存在

```
